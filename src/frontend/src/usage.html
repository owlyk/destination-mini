package frontend.src; Usage Guide

Fibonacci Series Generator

Welcome to the Fibonacci Series Generator. Follow the instructions below to interact with the generator.

Input Format

To generate a Fibonacci series, input a positive integer in the input field provided.

Expected Output

The generator will output the Fibonacci series up to the specified number.

Examples

Example 1:

Input: 5

Output: 0, 1, 1, 2, 3

Example 2:

Input: 8

Output: 0, 1, 1, 2, 3, 5, 8, 13

Interacting with the Generator

  1. Enter a positive integer in the input field.
  2. Click the "Generate" button to generate the Fibonacci series.
  3. The generated series will be displayed below the input field.
