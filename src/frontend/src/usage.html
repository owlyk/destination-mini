package frontend.src; Fibonacci Generator Usage

Welcome to the Fibonacci Generator!

How to Use:

Input Values:

To generate a Fibonacci sequence, simply input the number of terms you want in the sequence.

Interpreting the Generated Sequence:

The generated sequence will start with 0 and 1, and each subsequent number is the sum of the two preceding numbers.

Utilizing the Program:

You can use the generated Fibonacci sequence for various purposes such as mathematical calculations, pattern recognition, and more.

Examples:

Example 1:

Input: 5

Output: 0, 1, 1, 2, 3

Example 2:

Input: 8

Output: 0, 1, 1, 2, 3, 5, 8, 13
