Fibonacci Application Usage

Please enter a non-negative integer to calculate the Fibonacci number.

Input Requirements:

  • The input must be a whole number.
  • The input cannot be negative.

Expected Outputs:

  • The Fibonacci number corresponding to the input integer.
  • An error message if the input is invalid or negative.

Error: Invalid input. Please ensure you enter a non-negative integer.
