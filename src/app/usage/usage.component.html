How to use the Fibonacci web app

{{ msg }}

To use the app, just add the number of Fibonacci numbers you want to calculate to the end of the above URL.

Example format: /fib/

  • /fib/5 - First 5 Fibonacci numbers
  • /fib/20 - First 20 Fibonacci numbers
  • /fib/100 - First 100 Fibonacci numbers
  • /fib/2e6 - First 2x10^6 Fibonacci numbers
