How to Use the Fibonacci Calculator

Understanding the Fibonacci Sequence

The Fibonacci sequence is a series of numbers where each number is the sum of the two preceding ones, usually starting with 0 and 1. The sequence goes: 0, 1, 1, 2, 3, 5, 8, 13, 21, and so on.

Step-by-Step Instructions

  1. Open the Fibonacci Calculator application.
  2. Enter a positive integer in the input field.
  3. Click the "Calculate" button.
  4. The Fibonacci number corresponding to the entered integer will be displayed.

Examples

Example 1

Input: 5

Expected Output: 5

Example 2

Input: 10

Expected Output: 55
