Fibonacci Application Usage Instructions

Welcome to the Fibonacci application! To use this application, please follow the instructions below:

  1. Enter a positive integer as input.
  2. Submit the input to receive the Fibonacci number corresponding to that integer.
  3. If the input is invalid (e.g., negative numbers, non-integer values), you will be redirected to this usage instructions page.

Example:

  • Input: 10 → Output: 55
  • Input: -5 → Output: You will see this instructions page.

If you have any questions, please refer to the help section or contact support.
