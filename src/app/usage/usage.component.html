How to Use the Fibonacci Calculator

This endpoint does not process any user input or perform calculations. Please note that this is for informational purposes only.

  1. Open the Fibonacci Calculator application.
  2. View the results displayed on the screen.

Examples


Input: 5
Output: 0, 1, 1, 2, 3, 5

Input: 10
Output: 0, 1, 1, 2, 3, 5, 8

Notes

The Fibonacci sequence starts with 0 and 1, and each subsequent number is the sum of the two preceding ones.
