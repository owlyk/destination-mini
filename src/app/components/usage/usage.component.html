Fibonacci Calculator Usage Instructions

Welcome to the Fibonacci Calculator! Please follow the instructions below to use the application:

  1. Enter a non-negative integer in the input field.
  2. Click the "Calculate" button to receive the Fibonacci sequence up to the entered number.
  3. If the input is invalid, an error message will be displayed.

Error Messages:

  • If the input is not a number: "Please enter a valid number."
  • If the number is negative: "Please enter a non-negative integer."

Thank you for using the Fibonacci Calculator!
