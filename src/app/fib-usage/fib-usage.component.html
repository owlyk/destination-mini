Fibonacci Usage Instructions

Acceptable Input Formats

The Fibonacci function accepts the following input formats:

  • Integers (e.g., 0, 1, 2, 3)
  • Scientific notation (e.g., 1e3 for 1000)

Edge Case Handling

When using the Fibonacci function, please note the following edge cases:

  • Negative numbers are not accepted and will result in an error.
  • Input of zero will return the first Fibonacci number, which is 0.

Output Format

The output will display the Fibonacci numbers up to the specified input value. For example:

  • If the input is 5, the output will be: 0, 1, 1, 2, 3, 5
  • If the input is 10, the output will be: 0, 1, 1, 2, 3, 5, 8
