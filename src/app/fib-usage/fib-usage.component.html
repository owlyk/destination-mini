Fibonacci Usage Instructions

Acceptable Input Formats

The input must be a non-negative integer. Acceptable formats include:

  • Whole numbers (e.g., 0, 1, 2, 3)
  • Scientific notation (e.g., 1e3 for 1000)

Negative Numbers

Negative numbers are not valid inputs. Please ensure that your input is a non-negative integer.

Edge Cases

Consider the following edge cases:

  • Input of 0 returns the first Fibonacci number: 0
  • Input of 1 returns the second Fibonacci number: 1
  • Input of 2 returns the third Fibonacci number: 1
  • Input of 3 returns the fourth Fibonacci number: 2

Output Format

The output will display Fibonacci numbers corresponding to the input provided. For example:

  • Input: 5 → Output: 5
  • Input: 10 → Output: 55

Data Binding

Fetched data will be dynamically displayed using Angular's data binding features.
