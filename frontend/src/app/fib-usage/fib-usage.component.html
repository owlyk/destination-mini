Fibonacci Calculation Instructions

Input Format

Please enter a number for Fibonacci calculations. Acceptable input formats include:

  • Positive integers (e.g., 5)
  • Scientific notation (e.g., 1e2 for 100)

Valid Input Range

The input must be a positive integer. Negative numbers and zero are not valid.

Expected Output Format

The output will be a list of Fibonacci numbers up to the specified input number.

Error Handling

If the input is invalid, the following errors may occur:

  • Non-integer input: Please enter a valid integer.
  • Negative input: Please enter a positive integer.
  • Zero input: Please enter a positive integer.

Common Mistakes

Here are some examples of common mistakes and how to correct them:

  • Input: "abc" - Correction: Enter a valid number.
  • Input: "-5" - Correction: Enter a positive integer (e.g., 5).
  • Input: "0" - Correction: Enter a positive integer (e.g., 1).
