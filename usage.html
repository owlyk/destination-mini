Fibonacci Calculation Feature Usage

Example 1: Calculating Fibonacci Number

To calculate the nth Fibonacci number, make a GET request to the following endpoint:

/fibonacci?n=5

This will return the 5th Fibonacci number.

Example 2: Displaying Fibonacci Sequence

To display the first n Fibonacci numbers, make a GET request to the following endpoint:

/fibonacci-sequence?n=8

This will return the first 8 Fibonacci numbers in a sequence.

Example 3: Fibonacci Number in HTML Template

You can also embed the Fibonacci number directly in an HTML template using Flask:

        <html>
        <head>
            <title>Fibonacci Number</title>
        </head>
        <body>
            <h1>Fibonacci Number: {{ fibonacci_number }}</h1>
        </body>
        </html>

Make sure to render the template using Flask's render_template function.
